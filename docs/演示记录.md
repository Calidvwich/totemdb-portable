## 任务一

已在 Android 前端执行：

- 创建无 WHERE 的非严格 SelectDeputy。
- 普通 INSERT 不进入非严格代理。
- `INSERT ... INTO` 显式进入代理。
- DELETE 和 UPDATE 同步维护已关联代理元组。
- 未关联元组不影响非严格代理。
- 严格代理拒绝显式 `INTO`。
- 无关源类拒绝写入该非严格代理。

关键输出：

```text
ManualStudent
|name                |score               |
|Alice               |95                  |

Error: explicit INTO is not allowed for strict deputy StrictStudent
Error: ManualStudent is not a non-strict deputy of the source class
```

## 任务二

已在 Android 前端执行：

- `HAVING AVG(salary) > 5000`：保留 HR、IT。
- `HAVING AVG(salary) >= 3000`：保留 HR、IT、Sales。
- 创建 HighSalaryDept 时过滤 Sales。
- INSERT IT 10000：IT 更新为 avg=9000、cnt=3。
- INSERT Sales 9000：Sales 以 avg=6000、cnt=2 进入。
- DELETE IT id=4：IT 更新后仍保留。
- DELETE Sales 高工资成员：Sales 退出。
- UPDATE IT id=4 为 9500：IT 更新为 avg=8750。
- UPDATE HR id=2 为 3000：HR 退出。
- 将 HR 两名员工更新为 9000：HR 重新进入。
